## Course Structure

### Module 4: MCP Integration for PMs (6 lessons, ~2.5 hours)

**4.1 Introduction to MCPs** (20 minutes)
- What is Model Context Protocol?
- How MCPs connect Copilot to your PM tools
- Security, permissions, and authentication
- Time savings overview (10-15 hours/week)

**4.2 Confluence Integration** (30 minutes)
- Search and read Confluence pages
- Create and update documentation
- Automate PRD syncing
- Research synthesis workflows
- Weekly status updates
- **Includes:** PRD template example
- **Time saved:** 5-10 hours/week

**4.3 Azure DevOps** (30 minutes)
- Query work items and sprints
- Create and update user stories/bugs
- Sprint planning automation
- PR management and reviews
- Automated reporting workflows
- **Includes:** User story template
- **Time saved:** 2-4 hours/week

**4.4 Figma Collaboration** (25 minutes)
- Browse design files and frames
- Extract design specifications
- Design token management
- Reference designs in PRDs
- Design handoff automation
- **Note:** Read-only MCP access currently
- **Time saved:** 3-5 hours/week (when fully available)

**4.5 Miro Visual Collaboration** (25 minutes)
- Access and browse Miro boards
- Extract sticky notes and cards
- Synthesize workshop outputs
- Generate action items from boards
- Cross-board analysis
- **Time saved:** 83 min per workshop (80-90% reduction)

**4.6 Build HTML Presentations (BONUS)** (35 minutes)
- Create reveal.js presentations
- Pull data from all MCP tools
- Build interactive dashboards
- Add charts and visualizations
- ASOS brand styling
- Export to PDF and host online
- **Includes:** 3 complete templates
- **Time saved:** 2+ hours per presentation

## How This Course Works

### Interactive Teaching
1. Open a module's `SCRIPT.md`
2. Ask Copilot to follow the script
3. Copilot becomes your instructor
4. Practice with real exercises
5. Get immediate feedback
6. Build actual PM artifacts

### Learning Pattern
Each module follows:
1. **Introduction** - What and why
2. **Setup** - Configure the MCP tool
3. **Phases** - 8-10 progressive exercises
4. **Real Workflows** - 3+ practical PM scenarios
5. **Integration** - Connect to other modules
6. **Best Practices** - DOs and DON'Ts
7. **Practice Challenge** - Hands-on exercise
8. **Reflection** - Consolidate learning
